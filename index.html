Mastering Valorant

Insights into Agent Selection and Performance

This data analytics project aims to analyze player performance and agent abilities in the popular first-person shooter game, Valorant. The project uses data from two tables - one with player stats such as win percentage, KDA, and pick rate, and another with information about each agent's abilities. By combining and analyzing these datasets, the project aims to identify trends and insights that can help players make more informed decisions about agent selection and gameplay strategies.

PowerBI Dashboard for the inferences.

The project also includes a dashboard with visually appealing graphs and visuals that provide a comprehensive overview of player performance and agent abilities. The ultimate goal of this project is to help players improve their gameplay and achieve greater success in Valorant.
